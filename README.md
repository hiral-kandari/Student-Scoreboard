# Async Weather Tracker 🌦️

A modern weather tracking web application built using **React.js** and **Async JavaScript (Fetch API + Async/Await)**.  
This project allows users to search weather information for any city and view real-time weather updates in a clean and responsive UI.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- ☁️ Weather condition information
- 💨 Wind speed and humidity details
- ⚡ Async API handling using `async/await`
- 📱 Responsive design for all devices
- ❌ Error handling for invalid city names

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

- Async/Await

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/hiral-kandari/Async-Weather-Tracker.git
```

### 2. Navigate to project folder

```bash
cd Async-Weather-Tracker
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```
